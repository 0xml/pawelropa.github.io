code highlighting in ghost

Ghost blogging platform is great. It suits my needs almost perfectly as I don’t need a colossal blogging platform with millions of plugins (and security vulnerabilities) like Wordpress. The posts editor is top notch, really like the preview functionality. For Ghost to be perfect I needed to add two main features; comments and code highlighting. This post will be about the latter. There are multiple choices for the javascript code which highlights syntax.
